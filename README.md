# openEvery
一次性打开多个应用的小脚本

# 使用方法
打开项目路径 **src/src/applicationSrc.txt** 文件, 找到需要一键打开的文件的可执行文件, 右键该文件选择复制文件地址, 将地址复制到 **applicationSrc.txt** 文件中, 保存文件。然后打开 **openEvery.exe** 即可一键打开 **applicationSrc.txt** 中添加的应用程序。
